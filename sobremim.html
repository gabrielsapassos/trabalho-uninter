Sobre mim

Ultimamente tenho buscado aprender mais sobre o geral do mundo da programação, então decidi aprender React com JavaScript, Spring Boot e usar o MySQL. Gosto muito de criar estilos com CSS, mas também acho muito interessante o Backend dos sites e aplicativos. Venho estudando um pouco de Java e JavaScript para interligar o Backend com o Frontend.

No futuro pretendo aprender Rust, por ser uma linguagem na qual se precisa conhecer o que acontece também embaixo dos panos do computador e acaba te ensinando muito mais que uma linguagem de alto nível. Além de AWS para hospedar projetos também.
